# Story 16: Sentiment Tracking

Feature area: Future Integrations

Story:
As a moderator, I want sentiment trends over time, so that I can see if the community is improving.

Acceptance criteria:
- Sentiment metrics are stored per day.
- A trend chart or summary is available.

Feasibility rating: Medium

Justification:
Requires storage and periodic aggregation.

Implementation notes:
- Store daily aggregates only to reduce storage.
- Provide a summary view with a short time range.
